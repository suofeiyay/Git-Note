# 排序和检索算法

计算机中存储的数据执行的最常见的操作就是排序和检索。

#### 1.基本排序算法

##### 1.1 冒泡排序

```javascript
//比较相邻的数据，左大于右时互换
let arr = [10,8,3,2,2,4,9,5,4,3]
function bubbleSort(arr){
	let len = arr.length;
	for(let i=len-1;i>=2;i--){
        for(let j=0;j<=i-1;j++){
            if(arr[j] > arr[j+1]){
                let temp = arr[j+1];
                arr[j+1] = arr[j];
                arr[j] = temp;
            }
        }
    }
}
//时间复杂度 O(n2)，空间复杂度O(1)
```
